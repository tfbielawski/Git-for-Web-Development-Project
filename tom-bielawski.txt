Stretch Questions

    1. What is the difference between an inline element and a block element?
    Block elements start a new line, take up available width, have top and bottom margins
    Inline elements do not start new line, use only necessary width
    2. What happens when an element is positioned absolutely?
    The element is positioned relative to the nearest positioned ancestor. 
    3. How do I make an element take up only the amount of space it needs-
     but also have the ability to give it a width? By using the "auto" value
    4. Name 3 elements that are display block by default: <p> <h1> <address>
     -2 elements that are display inline by default: <span> <img> <a>
     -1 element that is display inline-block by default: .className { display: inline-block; }

    5. In your own words, explain the box model. 
    What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
    It's a CSS method of wrapping HTML elements in 4 different parts: content, padding, border, margins
    The box model is not "responsive"
